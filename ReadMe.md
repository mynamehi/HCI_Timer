## sceneの種類
1. timerScene  
タイマーページ(ホームページ)

2. SettingsScene    
設定ページ

3. DrawingScene   
コップを制作するページ

4. CupsScene  
作成したコップを選ぶことができる



## timerScene

メニュー
時間
コップ

### タイマ
- タイマー  
ポモドーロ法をを採用する。
勉強時間25分休憩5分の勉強法。
[表示]
00:00
勉強時間が終われば休憩タイマーが自動で更新する。

- コップ部分
時間を同期して水の量が変化する。
作成したものも表示できるようにする。

- メニュー部分
 ほかの設定などSceneに遷移する。

## DrawingScene
- 枠線ボタン
- 水ボタン
- 完成ボタン

完成ボタンを押すと
名前を決めるwindowを出して
- 決定ボタン
- キャンセルボタン
を出す

## CupsScene
コップを一覧で見ることができる
現在、選択しているコップの名前を一番上に表示

閲覧できる情報は
- 作った日
- なまえ
機能は削除とセット
